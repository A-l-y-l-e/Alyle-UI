Get Started

Install

npm install alyle-ui --save

Import module in appModule

{{"// Core module import { AlyleUIModule } from 'alyle-ui'; // Responsive module (Optional) import { ResponsiveModule } from 'alyle-ui/responsive'; // config theme const configAlyleUI = { primary: 'blue', accent: 'pink', other: 'red' } @NgModule({ ... imports: [ AlyleUIModule.forRoot(configAlyleUI), ResponsiveModule ], ... }) export class AppModule {}"}}
