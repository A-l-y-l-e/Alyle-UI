<h1 id="theming-components">Theming Components</h1>
<p>You can customize the appearance of a component by replacing styles or adding new styles.</p>
<h2 lyTyp="headline" gutter>How to customize a component</h2>

<p>We must keep in mind that when we extend a theme, styles can be replaced or added one over another.</p>
<p>For example if I declare the styles in the <code class="ttm ttx">minimal-dark</code> theme, and also declare in <code class="ttm ttx">GlobalVariables</code>, the style that will be rendered will be the one in Global variables. This is a default behavior, to render both styles you can use <code class="ttm ttx">StyleCollection</code>.</p>
