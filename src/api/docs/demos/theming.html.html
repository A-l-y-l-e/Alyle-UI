<h2 id="overwriting-the-variables-of-a-theme">Overwriting the variables of a theme</h2>
<p>
  <code class="ttm ttx">MinimaLight &amp; MinimaDark</code> are already created themes, you can edit and add new variables.
</p>
<p>
  To overwrite a theme, you must create a new class and add the property <code class="ttm ttx">name</code> where the value is equal to the name of the theme to be overwritten.
